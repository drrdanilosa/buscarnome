1. Introdução

DeepAlias Hunter Pro é uma extensão avançada para Firefox e Microsoft Edge que permite buscar usernames, nomes reais e termos específicos em diversas plataformas online, incluindo sites de conteúdo adulto, fóruns, redes sociais e deep web.

Esta ferramenta foi projetada para ajudar pessoas a verificarem se seus dados pessoais ou imagens estão sendo utilizados de forma não autorizada, ilegal ou fraudulenta na internet, especialmente em um contexto onde a IA pode facilmente gerar conteúdo falso e identidades fictícias.

Principais Recursos

  • Busca em mais de 220 plataformas diferentes
  • Cobertura de sites adultos, fóruns especializados e deep web
  • Variações inteligentes de username para maior abrangência
  • Análise de risco e confiança dos resultados
  • Suporte a proxy e rotação de user-agent para evitar bloqueios
  • Integração com Tor para acesso à deep web
  • Análise de imagens para detecção de conteúdo sensível
  • Integração com serviços OSINT externos
  • Exportação de resultados em múltiplos formatos

2. Instalação

Instalação no Firefox

  1. Extraia o arquivo ZIP da extensão para uma pasta em seu computador
  2. Abra o Firefox e digite about:debugging na barra de endereços
  3. Clique em "Este Firefox" no menu lateral
  4. Clique em "Carregar extensão temporária..."
  5. Navegue até a pasta onde você extraiu a extensão e selecione o arquivo manifest.json
  6. A extensão será carregada e aparecerá na lista de extensões temporárias

Dica: Para instalação permanente, você precisará empacotar a extensão e enviá-la para a Mozilla para revisão, ou usar o Firefox Developer Edition ou Nightly, que permitem a instalação permanente de extensões não assinadas.

Instalação no Microsoft Edge

  1. Extraia o arquivo ZIP da extensão para uma pasta em seu computador
  2. Abra o Edge e digite edge://extensions na barra de endereços
  3. Ative o "Modo de desenvolvedor" no canto inferior esquerdo
  4. Clique em "Carregar sem pacote"
  5. Navegue até a pasta onde você extraiu a extensão e selecione-a
  6. A extensão será carregada e aparecerá na lista de extensões

Importante: Ao instalar a extensão no Edge, certifique-se de usar o arquivo manifest_edge.json renomeado para manifest.json, pois há pequenas diferenças na implementação entre os navegadores.

3. Primeiros Passos

Interface Principal

Após a instalação, você pode acessar a extensão clicando no ícone da DeepAlias Hunter Pro na barra de ferramentas do navegador. A interface principal contém:

  • Campo de busca: Para inserir o username, nome ou termo a ser pesquisado
  • Opções avançadas: Configurações específicas para a busca atual
  • Área de resultados: Onde os resultados da busca serão exibidos
  • Barra de status: Mostra o progresso da busca atual
  • Opções de exportação: Para salvar os resultados em diferentes formatos

Realizando Sua Primeira Busca

  1. Clique no ícone da extensão para abrir o popup
  2. Digite o username ou termo que deseja buscar no campo de texto
  3. Clique no botão "Buscar" para iniciar a pesquisa
  4. Aguarde enquanto a extensão verifica as diversas plataformas
  5. Os resultados começarão a aparecer à medida que forem encontrados
  6. Ao final da busca, você verá um resumo com o número de resultados encontrados

Dica: Para buscas mais eficientes, use termos específicos e evite nomes muito comuns, que podem gerar muitos falsos positivos.

4. Busca Avançada

Opções Avançadas de Busca

Para acessar as opções avançadas, clique em "Opções avançadas" abaixo do campo de busca. Isso revelará configurações adicionais:

  • Incluir sites adultos: Ativa ou desativa a busca em plataformas de conteúdo adulto
  • Incluir deep/dark web (Tor): Ativa a busca em sites .onion e deep web (requer configuração do Tor)
  • Categorias prioritárias: Permite selecionar quais categorias de sites devem ser verificadas primeiro
  • Máx. variações por plataforma: Define quantas variações do username serão testadas em cada plataforma

Interpretando os Resultados

Cada resultado encontrado contém as seguintes informações:

  • Nome da plataforma: O site ou serviço onde o username foi encontrado
  • URL: Link direto para o perfil ou página encontrada
  • Pontuação de risco: Estimativa do nível de risco associado ao resultado (alto, médio, baixo)
  • Confiança: Probabilidade de o resultado ser realmente relevante
  • Variação: Qual variação do username foi encontrada

Exportando Resultados

Após concluir uma busca, você pode exportar os resultados em diferentes formatos:

  • JSON: Formato estruturado para análise técnica ou importação em outras ferramentas
  • CSV: Formato de planilha compatível com Excel, Google Sheets, etc.
  • HTML: Relatório formatado para visualização em navegador

Para exportar, basta clicar no botão correspondente ao formato desejado na parte inferior da área de resultados.

5. Configurações

Para acessar as configurações completas da extensão, clique em "Configurações" no rodapé do popup. Isso abrirá uma página de configurações com várias seções:

Configurações Gerais

  • Requisições Simultâneas: Controla quantas verificações são feitas em paralelo
  • Tempo de Cache: Define por quanto tempo os resultados são armazenados
  • Nível de Log: Controla a quantidade de informações de diagnóstico registradas

Configurações de Busca

  • Incluir sites adultos por padrão: Define o comportamento padrão para novas buscas
  • Incluir deep/dark web por padrão: Define se buscas na deep web são ativadas automaticamente
  • Categorias Prioritárias: Define quais categorias são verificadas primeiro
  • Máx. variações por plataforma: Define o número padrão de variações

Proxy e User-Agent

  • Habilitar Rotação de Proxy: Ativa o uso de múltiplos proxies para evitar bloqueios
  • Lista de Proxies: Permite adicionar servidores proxy para rotação
  • Habilitar Rotação de User-Agent: Alterna entre diferentes identificações de navegador

Conexão Tor

  • Habilitar Conexão Tor: Ativa o acesso a sites .onion e deep web
  • Proxy Tor: Configura o endereço do proxy SOCKS do Tor
  • Testar Conexão: Verifica se a conexão com a rede Tor está funcionando

Análise de Imagem

  • Habilitar Análise de Imagem: Ativa a detecção de conteúdo sensível em imagens
  • Provedor de API: Seleciona qual serviço usar para análise de imagens
  • Chave de API: Configura a chave para o serviço selecionado

Integração OSINT

  • Habilitar Integração OSINT: Ativa a conexão com serviços externos de inteligência
  • Chaves de API: Configura as chaves para serviços como Have I Been Pwned, DeHashed, etc.

Dica: Após fazer alterações nas configurações, clique em "Salvar Configurações" para aplicá-las. Algumas alterações podem exigir que você reinicie a extensão.

6. Recursos Avançados

Acesso à Deep Web via Tor

Para acessar sites na deep web e dark web (.onion), você precisa:

  1. Instalar o Tor Browser (disponível em torproject.org)
  2. Iniciar o Tor Browser para que o serviço SOCKS esteja disponível
  3. Nas configurações da extensão, habilitar "Conexão Tor"
  4. Verificar se o proxy está configurado corretamente (geralmente socks5://127.0.0.1:9050)
  5. Testar a conexão usando o botão "Testar Conexão Tor"
  6. Nas opções avançadas de busca, marcar "Incluir deep/dark web (Tor)"

Atenção: O acesso à deep web e dark web pode expor você a conteúdo ilegal ou perturbador. Use com responsabilidade e apenas para fins legítimos de investigação.

Configuração de Proxy

Para usar proxies e evitar bloqueios ou limites de taxa:

  1. Nas configurações, habilite "Rotação de Proxy"
  2. Adicione seus proxies na lista, um por linha, no formato host:porta ou host:porta:usuario:senha
  3. A extensão alternará automaticamente entre os proxies durante as buscas

Análise de Imagem

Para detectar conteúdo sensível em imagens encontradas:

  1. Nas configurações, habilite "Análise de Imagem"
  2. Selecione um provedor de API (Sightengine recomendado)
  3. Configure sua chave de API para o serviço selecionado
  4. Durante as buscas, a extensão analisará automaticamente imagens encontradas

A análise de imagem pode detectar:

  • Nudez e conteúdo adulto
  • Violência ou conteúdo perturbador
  • Presença de rostos (para identificar se suas fotos estão sendo usadas)

Integração com Serviços OSINT

Para verificar vazamentos de dados e presença em bases de dados comprometidas:

  1. Nas configurações, habilite "Integração OSINT"
  2. Configure as chaves de API para os serviços desejados
  3. Durante as buscas, a extensão consultará automaticamente esses serviços

Serviços suportados incluem:

  • Have I Been Pwned - Verifica vazamentos de e-mails
  • DeHashed - Busca credenciais vazadas
  • Intelligence X - Busca em bases de dados de inteligência

7. Perguntas Frequentes

Quais navegadores são suportados?

A extensão é compatível com Firefox (prioridade) e Microsoft Edge. Outros navegadores baseados em Chromium podem funcionar, mas não são oficialmente suportados.

A extensão armazena ou envia meus dados para algum servidor?

Não. A extensão funciona inteiramente no seu navegador e não envia dados para servidores externos, exceto quando você usa recursos específicos que requerem APIs externas (como análise de imagem ou integração OSINT).

Como posso saber se os resultados são precisos?

Cada resultado inclui uma pontuação de confiança que indica a probabilidade de ser um resultado verdadeiro. Resultados com alta confiança são mais prováveis de serem relevantes. Sempre verifique manualmente os resultados importantes.

Por que preciso do Tor Browser para acessar a deep web?

O Tor Browser fornece o serviço SOCKS necessário para rotear tráfego através da rede Tor, permitindo acesso a sites .onion e outros recursos da deep web. A extensão usa esse serviço para fazer as requisições.

A extensão pode remover conteúdo não autorizado que encontrar?

Não. A extensão apenas identifica onde seu username, nome ou imagens podem estar sendo usados. Para remover conteúdo não autorizado, você precisará entrar em contato com os administradores do site ou tomar medidas legais apropriadas.

Como obter chaves de API para os serviços externos?

Você precisará se registrar nos respectivos serviços:

  • Have I Been Pwned: haveibeenpwned.com/API/Key
  • Sightengine: sightengine.com/pricing
  • DeHashed: dehashed.com/pricing

A extensão é gratuita?

A extensão em si é gratuita, mas alguns recursos avançados (como análise de imagem e integração OSINT) podem requerer chaves de API de serviços externos que podem ter custos associados.

8. Solução de Problemas

A extensão não está encontrando resultados

  • Verifique sua conexão com a internet
  • Tente aumentar o número de variações por plataforma
  • Verifique se você não está sendo bloqueado por muitas requisições (considere usar proxies)
  • Tente termos de busca mais específicos

A conexão com Tor não funciona

  • Certifique-se de que o Tor Browser está instalado e em execução
  • Verifique se o endereço do proxy está correto (geralmente socks5://127.0.0.1:9050)
  • Reinicie o Tor Browser e tente novamente
  • Verifique se seu firewall não está bloqueando a conexão

A extensão está lenta

  • Reduza o número de plataformas verificadas usando categorias prioritárias
  • Diminua o número de variações por plataforma
  • Aumente o número de requisições simultâneas (se seu computador e conexão suportarem)
  • Verifique se o cache está habilitado para evitar verificações repetidas

Erros de API em serviços externos

  • Verifique se suas chaves de API estão corretas e válidas
  • Confirme se você não excedeu os limites de uso da API
  • Tente desabilitar temporariamente o serviço específico que está causando erros

A extensão desaparece após reiniciar o navegador

  • No Firefox, extensões carregadas temporariamente são removidas ao reiniciar. Você precisará carregá-la novamente ou criar uma versão assinada
  • No Edge, verifique se o modo de desenvolvedor ainda está ativado

Dica para desenvolvedores: Se encontrar problemas técnicos, verifique o console do navegador (F12 > Console) para mensagens de erro detalhadas que podem ajudar a diagnosticar o problema.
